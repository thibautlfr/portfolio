Un outil en ligne de commande pour deployer des sites statiques sur AWS S3 avec invalidation CloudFront automatique.

## Usage

```bash
cli-deploy init
cli-deploy push --env production
```

## Fonctionnalites

- Detection automatique du framework (Astro, Next.js, Vite)
- Upload incremental (ne pousse que les fichiers modifies)
- Invalidation CloudFront automatique
- Gestion multi-environnements
- Configuration via fichier `.deploy.toml`
