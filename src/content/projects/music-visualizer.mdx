Un visualiseur audio 3D qui reagit en temps reel a la musique jouee sur Spotify. Les shaders GLSL generent des formes geometriques qui evoluent avec le spectre audio.

## Etat du projet

Le projet est en cours de developpement. Le moteur de rendu et l'analyse audio fonctionnent, il reste a implementer :

- [ ] Connexion OAuth Spotify
- [ ] Mode VR via WebXR
- [ ] Editeur de presets visuels
- [x] Analyse FFT du signal audio
- [x] Pipeline de rendu Three.js
- [x] 5 shaders de base

## Architecture

Le signal audio est analyse via la Web Audio API (FFT), les donnees frequentielles sont passees aux shaders GLSL sous forme d'uniforms, et Three.js orchestre le rendu de la scene 3D.
