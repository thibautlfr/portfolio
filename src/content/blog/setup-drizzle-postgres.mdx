Drizzle ORM est devenu mon ORM de choix pour les projets TypeScript. Voici comment le configurer avec PostgreSQL.

## Installation

```bash
pnpm add drizzle-orm postgres
pnpm add -D drizzle-kit
```

## Schema

```typescript
import { pgTable, text, timestamp, uuid } from "drizzle-orm/pg-core";

export const users = pgTable("users", {
  id: uuid("id").primaryKey().defaultRandom(),
  name: text("name").notNull(),
  email: text("email").notNull().unique(),
  createdAt: timestamp("created_at").defaultNow(),
});
```

## Pourquoi Drizzle plutot que Prisma

- Les requetes sont du vrai SQL, pas un DSL custom
- Le bundle est plus leger (pas de moteur Rust embarque)
- Le type inference est plus naturel avec TypeScript
- Les migrations sont des fichiers SQL lisibles

## Ce que j'aurais aime savoir avant

Les relations ne sont pas gerees comme dans Prisma. Il faut penser en termes de jointures SQL plutot qu'en relations de modeles. Une fois le declic fait, c'est plus puissant et previsible.
